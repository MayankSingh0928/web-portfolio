# Mayank Singh Portfolio

Personal portfolio website for Mayank Singh, a full stack developer focused on React, Node.js, Django, APIs, Web3, and practical product engineering.

## Live Site

- Portfolio: https://web-portfolio-lyart-five.vercel.app
- GitHub repository: https://github.com/MayankSingh0928/web-portfolio

## Pages

- `index.html` - Main portfolio with hero, about, services, experience, featured work, blog preview, and contact form.
- `projects.html` - Additional public repositories and project cards.
- `blog.html` - Project notes and development stories.

## Run Locally

This is a static website. You can open `index.html` directly in a browser.

For a local server:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## Deployment

The project can be deployed on GitHub Pages, Vercel, Netlify, or any static hosting service.

For GitHub Pages:

1. Push the repository to GitHub.
2. Open repository Settings.
3. Go to Pages.
4. Select the `main` branch and root folder.
5. Save and wait for the site to publish.

## Notes

- The red square icon on project cards opens the deployed/live site when available.
- The yellow `Repo` button opens the source repository.
- Cards without a deployment show `Demo pending`.
